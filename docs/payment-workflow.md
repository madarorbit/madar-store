# دورة التحويل البنكي

1. ينشئ العميل طلباً ذرّياً بحالة `PENDING_PAYMENT`.
2. تعرض المنصة بنك القطيبي والحساب `488119337` والتنبيه المهني للحساب المؤقت.
3. يرفع العميل إثباتاً إلى تخزين خاص بعد تحقق النوع والحجم؛ تصبح الحالة `PAYMENT_UNDER_REVIEW`.
4. يراجع مسؤول مفوض الإثبات. القبول ينشئ `Payment` و`Download` وسجل تدقيق في معاملة واحدة؛ الرفض يسجل السبب ويسمح بإثبات جديد.
5. لا تنشأ روابط تنزيل قبل القبول.

إضافة بوابة دفع مستقبلية تتم عبر adapter يترجم نتيجة المزود إلى انتقالات `OrderStatus` نفسها.
