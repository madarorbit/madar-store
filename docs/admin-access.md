# الوصول الإداري

`SUPER_ADMIN_EMAIL=orbit.ops.digital@gmail.com` هو شرط bootstrap فقط، وليس تفويضاً في المتصفح. ينفذ seed آمن بعد تحقق البريد لإنشاء/ترقية الدور، ولا يقبل التسجيل العام أدواراً. يجب حماية `/admin/**` على الخادم، وتسجيل محاولة الرفض في `security_events`. تظل لوحة الإدارة الحالية غير متاحة كتشغيل إنتاجي حتى اكتمال المصادقة وقاعدة البيانات.
