# المصادقة والجلسات

المصادقة الدائمة **غير مفعلة بعد** لأن المشروع لا يملك قاعدة بيانات أو مزود بريد أو secret إنتاجي. لا يجوز وضع نموذج تسجيل يوهم العميل بأن حسابه محفوظ قبل تنفيذ ذلك.

عند التنفيذ: استخدم Argon2id لكلمات المرور، رموز تحقق/استعادة عشوائية مخزنة كبصمات فقط، rate limiting لكل IP والبريد، ورسائل خطأ عامة. تتحقق كل Route Handler وServer Action من الجلسة وملكية المورد. تتضمن الجلسات تاريخ انتهاء وجهازاً مبسطاً، وتستطيع صفحة الأمان إنهاء الجلسات الأخرى.
